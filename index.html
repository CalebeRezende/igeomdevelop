Interações no iGeom

Este documento detalha as principais interações disponíveis no aplicativo iGeom, incluindo informações para desenvolvedores e funcionalidades gerais.

Menu Lateral - Ferramentas

O menu lateral contém ferramentas que podem ser usadas para manipular o canvas. Exemplo:

  • Botão Ponto: Cria pontos no canvas. Está localizado no elemento <button id="btn-point">.
  • Status: Um indicador de status exibido no elemento <span id="status">.

Menu Superior

O menu superior permite acesso a diversas funcionalidades:

  • Arquivo: Permite abrir e salvar documentos. Os botões são representados por:
    • <button id="open">Open</button>
    • <button id="save">Save</button>
  • Editar: Opção de edição disponível, mas não detalhada no código fornecido.

Documentos

Os documentos criados podem ser visualizados e gerenciados por meio do painel de arquivos:

  • Documento Selecionado: Indicado pelo elemento <div id="selected-document">.
  • Novo Documento: Criado pelo botão identificado por <button class="new-file">+</button>.

Ferramentas Avançadas

Além das ferramentas básicas, o iGeom oferece funcionalidades avançadas como:

  • Segmentos de Reta: Permite criar segmentos de reta entre dois pontos.
  • Medição de Ângulos: Ferramenta para medir ângulos entre retas.
  • Cálculo de Áreas: Determina a área de polígonos definidos no canvas.

Configurações

Personalize o ambiente de trabalho para maior produtividade:

  • Temas: Alterne entre modos claro e escuro.
  • Unidades de Medida: Configure unidades como centímetros, metros ou pixels.
  • Grid Snapping: Ative para alinhar elementos ao grid.

Exemplos Práticos

Veja como utilizar o iGeom para resolver problemas geométricos:

  1. Construção de Triângulos:
    • Selecione dois pontos no canvas e utilize a ferramenta de triângulo equilátero para completar o terceiro vértice.
    • Utilize a ferramenta de medição para verificar se todos os lados possuem o mesmo comprimento.
  2. Divisão de Segmentos:
    • Selecione dois pontos para formar um segmento.
    • Utilize a ferramenta "Ponto Médio" para encontrar o ponto central do segmento.
    • Verifique as coordenadas do ponto médio e arraste os pontos iniciais para observar a atualização dinâmica.
  3. Interseções:
    • Desenhe duas retas utilizando pontos de início e fim.
    • Use a ferramenta de interseção para identificar o ponto onde as retas se cruzam.
    • Verifique se o ponto de interseção permanece consistente ao mover as retas.
  4. Construção de Polígonos:
    • Selecione pontos no canvas e conecte-os sequencialmente para formar um polígono.
    • Calcule a área utilizando a ferramenta de "Cálculo de Áreas".
    • Certifique-se de fechar o polígono corretamente antes de realizar medições.
  5. Aplicações em Sala de Aula:
    • Simule problemas geométricos como construções de circunferências inscritas e circunscritas.
    • Explore conceitos de geometria analítica, como inclinação e distância entre pontos.
    • Permita que os alunos testem interativamente teorias geométricas no canvas.

Para Desenvolvedores

Informações técnicas sobre o desenvolvimento do iGeom:

  • Scripts Utilizados:
    • jQuery: Manipulação de DOM e eventos.
    • Bootstrap: Estilização e layout responsivo.
    • Konva: Manipulação do canvas para desenho de formas e interações visuais.
  • Estrutura de Arquivos: O projeto inclui:
    • index.html: Arquivo principal para interface.
    • app.css: Estilizações específicas.
    • main.js: Scripts personalizados para interações e lógica.

Depuração e Logs

Ferramentas para monitorar e depurar o sistema:

  • Console de Logs: Exibe informações sobre interações e erros.
  • Modo Debug: Habilite para visualizar detalhes adicionais durante o uso.
  • Exportação de Logs: Salve os logs em um arquivo para análise posterior.
